## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- MongoDB 5+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/htf1389/physics-education-platform.git
cd physics-education-platform

# Install dependencies
npm install

# Create .env file
cp .env.example .env

# Update .env with your configuration
# Then run migrations
npm run db:migrate

# Start development server
npm run dev
```

### Development

```bash
# Start development (Frontend + Backend)
npm run dev

# Frontend only (http://localhost:5173)
npm run dev:client

# Backend only (http://localhost:5000)
npm run dev:server
```

### Production Build

```bash
# Build for production
npm run build

# Start production server
npm start
```
